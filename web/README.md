# Web app (`web/`)

This directory contains the **Channel Explorer** frontend. See the **[root README](../README.md)** for product context and features.

**Quick start:** `npm install` → `npm run dev`.

**Sitemap:** after changing static routes, run `npm run sitemap` to regenerate `public/sitemap.xml` (uses `VITE_SITE_URL` or defaults to production origin).
